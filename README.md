# Todolist_with_Flask

A simple todo list application with CRUD (Create, Read, Update, Delete) functions using Flask and SQLAlchemy.

## Features
- Add new tasks
- View all tasks
- Edit tasks
- Delete tasks

## Technologies Used
- Python
- Flask
- SQLAlchemy (SQLite as database)
- Bootstrap (for UI styling)
